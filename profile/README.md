## Main Feature
#### (1) 우리 가게 보고서
- 원하는 주기의 매장 보고서 생성
- 해충 이동 경로 이미지, 해충명, 해충 특징, 방역 체크리스트, 추천 약품 제공

- '오늘의 Issue', '궁금해요', '공유해요' 게시판 기능
